处理“权力斗争”（Dealing with power struggle）

与家长和成年人进行“权力斗争”是青少年成长中的正常现象，因为在这个阶段，年轻人开始表现出对独立和自主的需求。因此，他们更愿意质疑权威——尤其是父母的。当面对自伤等问题时，这类“斗争”更易激化，由于自伤产生的恐惧和掌控局面的尝试会导致成年人/父母以及孩子均处于非常紧张的状态。虽然不可能避免所有的“权力斗争”，但成年人可以通过了解一些基本原则来最小化“权力斗争”。

怎样才能知道何时发生了“权力斗争”？
#

当你的孩子拒绝做你要求做的事情、拒绝遵守你设定的规则、或者拒绝参加你希望他（或她）参加的活动时，“权力斗争”就已经开始了。多数情况下，这种对抗与要求本身关系不大，更多是为了拥有控制感或权力。父母或其他关心孩子的成年人很容易陷入“权力斗争”当中，特别是当我们认为自己的要求是合理的，或已经与孩子意见达成一致时。尽你所能不卷入“权力斗争”是避免它的最好办法。

策略一：尽早脱离
#

要想避免“权力斗争”引发争论，最好的方法就是不要参与其中。当你意识到“斗争”开始的那一刻就是开始尝试脱离它的时刻。这并不是妥协，而是给自己一些时间，使自己能在冷静的、没有过多强烈情绪的情况下想清楚该如何应对孩子的反抗。如果年轻人看上去就要和你吵起来了，保持语气和声音平静平和（对化解冲突）大有裨益。如果感觉到自己的情绪要上来了，那就承认并接受这种感受，但不要冲动地回应。记住：你有时间。多数情况下，以后还会有机会讨论这件事的。你也可以让年轻人在没有冲动行为的情况下化解掉负面情绪。让年轻人知道你并没有要忽视他或她，只是在倾听并想花时间去思考这一切。记住，一个巴掌拍不响，你可以简单地决定你不参战！

策略二：创造双赢局面
#

当你冷静下来时，试着问问自己，在这种情况下，什么可以让你们双方得到一些你们各自想要的东西？为了获得真正对你重要的，哪些要求在你看来是毫无斡旋余地的，哪些是可以妥协的？自己先想清楚，准备好了以后再找年轻人谈。你应该给他 / 她选择的余地，以使他 / 她能得到想要的，同时也清晰地阐明你所看重的是什么，这样可以提供一个可操作的中间地带，使每个人都能获得他们想要的东西。如果一个年轻人可以选择何时和如何履行义务，他或她在你所认可的框架下就拥有更多的自主权和决策权。

战略三：协作
#

试着把你的年轻人当作谈判过程中的一个伙伴。在你们都能平静对话的时候，如果你能创造机会，和他 / 她就你们双方所看重的东西谈谈，你会发现这真的很有用。这些交谈的目的可以是就期望和需求达成一致意见。当然，如果期望未能达成一致的话，也可以尝试就结果达成。谈话可以具体到自伤行为，比如双方可以接受的手法和工具，也可以涉及其他可能引爆父母或孩子强烈情绪的话题。对不履行责任导致的后果达成清楚、一致的意见也有助于避免争论不休的“权力斗争”。当后果已无可避免时，该意见至少有助于指导下一步工作。考虑采用“积极的后果”（例如帮助他人或做其他形式的服务）来代替拿走东西或限制自由。“积极的后果”有助于培养必要技能，并在成年人和年轻人之间产生积极的感情。在“权力斗争”开始时可以提到这些后果，提醒你的年轻人他或她参与了这些后果的讨论制定，并不断帮助他/她明白应该对自己的行为负责，如此有助于制止或至少更快平息“权力斗争”。

策略四：耐心和坚持
#

请记住，这种“斗争”是正常的，并实际上是孩子健康成长的标志。你们可以一起尝试为孩子成长过程中的自主、独立和对边界的探索（limit-testing）留有空间，找出避免冲突的沟通方式。有些时候你可能会觉得自己很失败，或觉得无论你自己多么合理，你的孩子都会选择最大程度地去对抗你而非避免对抗。在这些时候，成年人需要做的——这同样也是你们成长的机会——是减轻自身压力，相信时间和经验会帮助年轻人成熟起来。放慢速度、只观察而不行动、不期待快速产生效果、知道自己有时间，然后你需要花时间冷静地评估最佳的行为方式，这些都是非常有用的技巧。这样做也有助于你的孩子——他是通过观察你来学习的——培养应对压力、冲突和强烈负面情绪的技能。

策略五：感恩
#

对那些与自伤或自伤恢复期的年轻人共事的成年人来说，“权力斗争”会造成独特的挑战。担心你正常设定的界限会导致 ta 自伤是很常见的，并且会令人感到困惑难受。提醒你自己和你的年轻人，要为以往度过的艰难时刻而感恩。每次鼓励都对所有人很有帮助，它提醒人们有些事情完成得已经很好了，即使并不完美。
