NSSI ReadMe

2020-08-06
NSSI

我想了很久，最后还是决定在博客里开设这样一个分类，用来存储有关 NSSI （aka 非自杀性自伤行为，俗称自残）的翻译文章。目前来看，这些翻译页面的原文会来自 康奈尔大学的网站，参阅这里，他们允许在列明来源的情况下转载资源。其他地方如果有很好的资源我也会试着去做的。

熟悉我的人应该知道我本体是高中生，且不是国际生，英语水平破破烂烂也就将将够用，显然不是搞这个翻译工作的最佳人选。我会列出每一篇文章的原文页面，如果有朋友有这个时间和精力能帮我捉捉虫的话感激不尽。

之所以我想来做这件事，是因为——很多人不知道，我曾经，并且如今也深陷在 NSSI 的泥潭里。我的小臂上有裁纸刀留下的灰色疤痕，最早的一条来自初中二年级，至今为止，天冷时仍能明显看出。在和家长产生矛盾后，我很难“像个正常人那样”和他们争吵，而会下意识地拿各种身体部位撞墙。按照 Recovery 一栏列出的标准，我可能是在 preparation 那一步里面吧，爬到 secure 估计需要远远不止五年。

但我是幸运的。我妈妈知道我有这个问题，她也比较能理解我：虽然她很生气，但是能理解我。我的同学和老师们都很棒，不提这个秘密，他们都是非常暖心的人，我没有遇到过任何类似校园欺凌这样的事。我的英语水平至少将将够用，我能读这些研究成果、戒除策略和成功摆脱 NSSI 的经历分享，更好地完成自救。而且因为有了这么多的社会支持、家庭支持和他人帮助，我幸运地没有发展到很严重的地步（容貌损毁、肢体残缺等）。

然后我发现根据康奈尔大学的研究，自伤在青少年群体中的发生率相当高。近年来的一项研究通过分析世界范围内多个的数据，估算出：青少年自伤的发生率为 17.2%，年轻成人是 13.4%，成年人 5.5%。虽然自伤是一种相对私密的行为，因而难以准确统计，但这个数据至少提示我们：NSSI 的发生率比我们想象的要高。

所以对我来说，这项工程与其说是在尝试拯救他人，不如说是在尝试拯救我自己。我不想再沉沦于此。我不知道这么做到底有没有用，但昨天晚上当我在床上辗转反侧的时候我想明白了：出于虚荣心也好，出于无谓的理想主义也好，既然想翻译，而我又恰好折腾出了一整个可以任我挥洒的博客，那就去翻好了。

这些翻译资料很可能不全，很可能翻译得不好。但我保证我会去学翻译的方法和原则，会尽力去翻得准确通顺。再一次，如果我翻出了问题，请务必通过底下的 contact 找我。

祝这个世界再无 NSSI。愿我们一同超越悲伤，找到欢乐，或至少找到自由。
