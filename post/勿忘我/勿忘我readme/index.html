勿忘我

《勿忘我》是一个原创小说。

ABO设定，星际战争背景，或许算是软科幻。

在必要的位置可能出现（并不准确的）性描写、暴力描写，可能有主要角色死亡。章节前会有预警，觉得自己接受不了的话请不要看。按 AO3 那种分级法的话，应该是 Mature 的作品吧虽然原创不可能上 AO3 哈哈哈。

作者对真实世界中的战争知之甚少，战术及战略描写很可能有不合理之处。如果您对这方面有特殊的爱好和要求，请斟酌考虑，自行避雷别来打我。

CP性取向都是男女，但有女A男O等非主流配置。不会有生子，作者尽可能避免恋爱脑。目前 NE 已经完结，TE 未完结并可能咕咕很久。
